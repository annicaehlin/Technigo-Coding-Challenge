Your interactive buddy

JavaScript is a widely used programming language today. It adds dynamic behaviour to web pages, enabling them to be interactive. You can use JavaScript to program animations, clickable buttons, even games. Those pop up menus on web pages? That's JavaScript. Those hints that tell you you haven't supplied a valid e-mail address when submitting a form? Also JavaScript - used to validate the input in form fields.

Originally JavaScript was developed for use on the client-side, meaning it executed only in web browsers. These days JavaScript is also used on the server-side, running scripts to make dynamic page content before the page is sent to the web browser, for example. JavaScript is also used in web analytics, tracking a users' behavioural pattern to show them personalized content, for ad tracking and so on.
